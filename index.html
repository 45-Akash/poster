Programming Fundamentals Syllabus

A well-structured and beginner-friendly syllabus to master coding from scratch.

Hands-on: After every concept, we will solve example questions on that concept.

  • Introduction to programming: what code is and how computers run it
  • Thinking in steps: flowcharts and understanding code flow
  • Installation and IDE setup (compiler, editor, extensions)
  • Your first program: "Hello, World" + code structure, input and output
  • Variables, keywords and data types
  • Operators
    • Arithmetic, comparison, logical
    • Assignment, increment/decrement, ternary
    • Bitwise (and when to use them)
  • Strings: creation, methods/operations, formatting, common pitfalls
  • Math utilities: standard functions, precision and rounding
  • Decision making: if/else
  • Multi-way branching: switch–case
  • Loops: while, for, do–while
  • Loop controls: break and continue
  • Arrays: 1D/2D arrays, traversal, common patterns (searching, min/max)
  • Declaring vs defining, parameters and return types
  • Scope and lifetime, pass-by-value vs pass-by-reference
  • Base and recursive cases, call stack intuition
  • Tracing recursive execution, when to choose recursion
  • Object class
  • Four pillars of OOP:
    • Encapsulation
    • Abstraction
    • Inheritance
    • Polymorphism
  • Exceptions: try/catch, throwing and catching
  • Cleanup patterns (RAII) as the C++ alternative to finally blocks
  • Plan the app: requirements, features and milestones
  • Implement core features using what you learned
  • Test, iterate and polish; final presentation

Data Structures Syllabus

A comprehensive course designed to master data structure concepts essential for cracking technical interviews.

Implementation focused: Learn to build and use each data structure from scratch.

  • What are data structures and why do we need them?
  • Abstract Data Types (ADT) vs. Concrete implementations
  • Time complexity and space complexity analysis
  • Big O, Big Omega, and Big Theta notation
  • Choosing the right data structure for your problem
  • Advanced array techniques: two-pointer, sliding window
  • Matrix traversal patterns and manipulation
  • Multi-dimensional arrays and their applications
  • String manipulation and common operations
  • Common interview patterns and problem-solving strategies
  • Memory basics: address, value, and pointer variables
  • Pointer operators: & (address-of) and * (dereference)
  • Pointer arithmetic and arrays
  • Pointers with functions (pass-by-reference), double pointers (intro)
  • Common pitfalls: null/dangling pointers
  • Types: singly, doubly, and circular linked lists
  • Operations: insertion, deletion, reversal, and traversal
  • Fast & slow pointer technique (cycle detection)
  • Merging and sorting linked lists
  • Common patterns: finding middle, removing nth node, palindrome check
  • LIFO principle and stack operations (push, pop, peek)
  • Array-based and linked list-based implementations
  • Applications: parentheses matching, expression evaluation
  • Infix, prefix, and postfix notation conversions
  • Next Greater Element and related problems
  • Monotonic stack technique
  • FIFO principle and queue operations (enqueue, dequeue)
  • Circular queue and its advantages
  • Deque (double-ended queue) and its applications
  • Priority Queue: concept and use cases
  • Implementation using arrays and linked lists
  • Real-world applications: task scheduling, BFS traversal
  • Hash functions and collision handling
  • Hash Maps and Hash Sets
  • Chaining vs. open addressing
  • Common patterns: frequency counting, two-sum variants
  • Load factor and rehashing
  • Tree terminology: root, nodes, leaves, height, depth
  • Binary Trees: properties and types (complete, perfect, balanced)
  • Tree traversals: Inorder, Preorder, Postorder, Level-order
  • Binary Search Trees (BST): insertion, deletion, searching
  • Self-balancing trees: AVL trees, Red-Black trees (concepts)
  • Heap data structure and Priority Queue implementation
  • Trie (Prefix Tree) for string problems
  • Common patterns: lowest common ancestor, path sum, tree construction
  • Graph terminology: vertices, edges, directed vs. undirected, weighted graphs
  • Graph representations: adjacency matrix vs. adjacency list
  • Graph types: simple, multigraph, complete, connected, cyclic, acyclic (DAG)
  • Graph traversals: DFS (Depth-First Search) and BFS (Breadth-First Search)
  • Cycle detection in directed and undirected graphs
  • Connected components and strongly connected components
  • Common patterns: island problems, path finding, graph coloring
  • Real-world applications: social networks, maps, dependencies
  • Segment Trees: range query optimization
  • Fenwick Trees (Binary Indexed Tree)
  • Disjoint Set Union (Union-Find) with path compression
  • Sparse Tables for range minimum queries
  • Problem-solving patterns using advanced structures
  • When to use each advanced data structure
